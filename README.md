# Spring Examples

This repository contains a collection of example Spring Boot applications,
demonstrating various features, best practices and recipes for building
applications with the Spring Framework.

## Project Structure

Each top-level directory contains a separate example project, meant to be
built and run independently.

Instructions for building and running each example is to be found in its own
README.md.

### Prerequisites

You will need the following installed on your machine:

- Java Development Kit (JDK) 21 or higher
- Appropriate build tools for each project (e.g., Maven 3.6.3+ for Maven
  projects, Gradle for Gradle projects)
